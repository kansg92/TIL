# day54__Cloud

## 1. NCP(Naver Cloud Platform)

### Cloud

- IAAS(인프라스트럭처 서비스 제공)
  - 서버, 보안 장비를 제공하며 보통 유료임!
  - 비용을 제공하고 서비스를 받음.
  - 우리는 리눅스(Linux) 서버 기반으로 인프라를 구축 예정임.
  - Linux OS - JDK, Mysql, Web Server 를 설치 예정임.
- PAAS(플랫폼 서비스 제공)
  - Naver에서 제공하는 AI Platform을 사용 예정임.
  - 번역, 이미지 분석, 챗 봇 등....
  - 
- SAAS(소프트웨어 서비스 제공)

위 3개를 Cloud를 제공하며 우리 수업에서는 IAAS,PAAS를 이용.
