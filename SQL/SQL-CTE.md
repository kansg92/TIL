# SQL - CTEs

Common Table Expressions



```sql
WITH 뉴 테이블 이름 AS (테이블 만들 쿼리문)
```

서브쿼리 대안, 둘의 성능차이는 아직 논쟁이 많다

다만 서브쿼리의 가독성보다 좋고, 디버깅하기엔 더 유리하기 때문에 최근 CTEs를 사용하는것을 권한다.
